    FlightJS adalah framework javascript yang memiliki konsep yang agak berbeda dengan beberapa Framework javascript lainnya. Beberapa pendekatan utamanya adalah:

    1. Menitik beratkan komponen yang terhubung erat dengan DOM.

      Konsekuensi dari orientasi terhadap DOM ini adalah:

      1. Menjadikan DOM Event API sebagai jalur komunikasi event antar komponen.
      2. Tiap komponen harus diinisiasi terhadap sebuah element di DOM.

    2. Berbagi kode memanfaatkan mixin.

    3. Menyediakan API untuk Aspect Oriented Programming agar komponen bisa lebih fleksibel.
